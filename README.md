# COMP2068-TS-ASSESSMENT
This repo contains a hands-on assessment for COMP2068 in TypeScript.

## Instructions

Follow these steps to complete the assessment:

### 1. Download this Repository
Download or clone this repository to your local machine.

### 2. Upload to Your GitHub Repository
You have two options:
- **Option A**: Upload this code to an existing GitHub repository
- **Option B**: Create a new GitHub repository and upload the code there

### 3. Invite Collaborator
Send a collaboration invite to **eduardojaime** on your GitHub repository so your work can be reviewed.

### 4. Read Setup Instructions
Read through [SETUP.md](SETUP.md) as your first step to understand the project setup and requirements.

### 5. Complete the Coding Tasks
Complete all coding tasks in [in-class-assessment.ts](in-class-assessment.ts).

**Important**: Do not create any additional files other than your .env file. All your work should be done in the existing `in-class-assessment.ts` file.

Good luck! 🚀
